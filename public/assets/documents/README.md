# Public document library

Only reviewed, publication-approved documents belong in this directory. Never add
client materials, transaction data, private account information, unpublished figures,
or documents containing personal data.

- `cv/` — current public CV
- `handbook/` — public Arima Finance Engine handbook
- `research/` — author-approved research PDFs
- `investment-memorandum/` — public-safe extracts only
- `presentations/` — approved presentation previews
- `case-studies/` — synthetic or explicitly approved case studies

Pages must display **Available soon** until the referenced file actually exists.

Public flagship project

Arima Finance Engine

A modular Python framework for market-analysis orchestration, deterministic trade-lifecycle simulation and risk-management architecture.

GitHub RepositoryPublic Release v0.1.0

Executive overview

The public project demonstrates how configuration, market-data, analysis, replay, risk-plan, lifecycle, notification and broker concerns can be separated into reviewable components.

Public/private boundary: “The public repository is a sanitised architectural demonstration. Proprietary production logic and operational configuration remain private.”

Problem addressed

Financial-market software becomes difficult to validate when data access, decision logic, lifecycle state and external execution are tightly coupled. The public architecture shows how explicit boundaries support deterministic testing and controlled failure handling.

Design principles

  • Explicit configuration profiles with fail-closed selection
  • Deterministic, sequential replay over synthetic data
  • Single-purpose interfaces for risk plans and lifecycle state
  • External notification and broker transports disabled in the public demo
  • Credential-free CI and reproducibility-focused tests

Public capabilities

Pipeline architecture

Seven deterministic demonstration phases with non-trading outputs.

Reliability boundaries

Controlled missing-dependency behavior and disabled external transports.

Replay & lifecycle

Chronological replay and explicit lifecycle transition examples.

Public validation

Compilation, formatting, linting, tests and GitHub Actions.

Handbook and documentation

The repository README, architecture diagrams, demo guide, security policy and roadmap form the current public handbook. A standalone PDF handbook is Available soon.

Limitations

  • No production strategy, thresholds, scoring weights or risk caps
  • No live market-data provider, scheduler or broker execution
  • No published performance or profitability claim
  • Not a production trading system and not financial advice
